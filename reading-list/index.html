Reading List

A collection of books, papers, and resources that I'm currently reading, have completed, or plan to read. This list reflects my academic interests in mathematics, computer science, and formal methods, and serves as a reference for ongoing learning and research.

  • Introduction to Algebraic Number Theory
    Ian Stewart and David Tall (2020) Book Completed
    Comprehensive introduction to algebraic number theory with clear explanations of fundamental concepts
  • Software Foundations
    Benjamin Pierce et al. (2021) Online Completed
    Interactive textbook on formal verification using Coq
  • Formal Methods in Software Engineering
    Michael Jackson (2019) Book In Progress
    Essential reading on applying formal methods to software development
  • Programming Language Foundations in Agda
    Philip Wadler and Wen Kokke (2020) Online In Progress
    Interactive textbook on programming language theory using Agda
  • Category Theory for Programmers
    Bartosz Milewski (2019) Book Planned
    Category theory explained for programmers with practical examples
  • Homotopy Type Theory: Univalent Foundations of Mathematics
    The Univalent Foundations Program (2013) Book Planned
    Foundational work on homotopy type theory and univalent foundations
  • Type Theory and Formal Proof
    Rob Nederpelt and Herman Geuvers (2014) Book Planned
    Introduction to type theory and its applications in formal verification
  • Elements of Algebraic Geometry
    Robin Hartshorne (1977) Book Reference
    Classic graduate-level textbook on algebraic geometry
  • Real Analysis: Modern Techniques and Their Applications
    Gerald Folland (1999) Book Reference
    Comprehensive treatment of real analysis with applications
  • The Art of Computer Programming
    Donald Knuth (1997) Book Reference
    Classic comprehensive reference on computer programming algorithms
