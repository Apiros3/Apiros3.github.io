Reading List

A collection of books, papers, and resources that I'm currently reading, have completed, or plan to read. This list reflects my current / previous interests (from September 2025), but also acts as a memo for things I can read when I have free time. I have written summaries for certain papers, which should be on the notes page.

  • A Modal Analysis of Staged Computation
    Rowan Davies and Frank Pfenning (2001) Paper In Progress
    I've read between Sections 1 through 4. It's about how intuitionistic S4 gives a nice framework for analyzing computation stages. Our main focus was trying to understand the translation between the explicit and implicit system. I (should) have corresponding notes about this paper on this page.
  • Constructing Quotient Inductive-Inductive Types
    Ambrus Kaposi, Andras Kovacs and Thorsten Altenkirch (2019) Paper In Progress
    This is the first paper I read during my internship with Ambrus Kaposi. There was a lot of prerequities I was missing at the time, so the notes accompanying this is quite all over the place, but hopefully comprehensive nontheless
  • Homotopy Type Theory: Univalent Foundations of Mathematics
    The Univalent Foundations Program (2013) Book In Progress
    Slowly progressing through the book. We have an unofficial reading group at Oxford with individual students where we sometimes get along and talk about the contents. I (should) have corresponding notes on this page if you want to see what I've read through.
  • Introduction to Categories and Categorical Logic
    Samson Abramsky and Nikos Tzevelekos (2011) Book In Progress
    Clean lecture notes for the Categories, Proofs and Processes course at Oxford. I like to learn pure categorical concepts from a mathematical perspective, but I am interested in the CS applications. I'm reading between Chapters 1.6 to 1.8 right now.
  • Cellular Cohomology in Homotopy Type Theory
    Ulrik Buchholtz and Kuen-Bang Hou (2020) Paper Planned
    Hard paper, there's still a few things I'm missing but it is a paper that looks extremely interesting and I do want to get back to this sometime soon.
  • Linear Logic (in the Stanford Encyclopedia of Philosophy)
    Stanford (2006) Online Planned
    Part of the reading list for my Master's Thesis. Looks like a good first read on what Linear Logic is.
  • Sheaves in Geometry in Logic - A First Introduction to Topos Theory
    Saunders Mac Lane and Leke Moerdijk (2015) Book Planned
    As the title mentions. Not too sure when I can start reading this, but was recommended quite a while ago, so should probably get started.
  • The Blind Spot - Lectures on Logic
    Jean-Yves Girard (2011) Book Planned
    Has all the good logic stuff, I plan to read by section, currently quite interested in the Part III Linear Logic bit
  • Truncation Levels in Homotopy Type Theory
    Nicolai Kraus (2015) Thesis Planned
    Has a bunch of interesting results on propositional truncation.
  • Categorical Semantics of Linear Logic
    Paul-Andre Mellies (2009) Book Reference
    Part of the reading list for my Master's Thesis.
